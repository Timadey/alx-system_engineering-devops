# System engineering & DevOps

Module of **System engineering** and **DevOps**, carried out during **Full Stack Software Engineering studies** at **[Alx SE](https://www.alxafrica.com/)**.

## Technologies
* Scripts written in Bash 5.1
* Tested on Ubuntu 20.04 LTS
* Puppet 3.8

## Projects
All of the following folders are projects done during the studies:

| Project name | Description |
| ------------ | ----------- |
| [`0x00-shell_basics`](https://github.com/Timadey/alx-system_engineering-devops/tree/master/0x00-shell_basics) | It aims to learn about basics commands, navigation, files and directories in **Shell** |
| [`0x01-shell_permissions`](https://github.com/Timadey/alx-system_engineering-devops/tree/master/0x01-shell_permissions) | It aims to learn about man pages, permissions (owner, group and other) of files and directories in **Shell** |
| [`0x02-shell_redirections`](https://github.com/Timadey/alx-system_engineering-devops/tree/master/0x02-shell_redirections) | It aims to learn about how to handle standard input and output and how to combine commands and filters with redirections in **Shell** |
